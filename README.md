## Getting started

### Requirements

* [nodejs / npm](http://nodejs.org/)
* A running [Mongodb](http://www.mongodb.org/downloads)


### Launch the app
````
npm install
npm start
````
Browse to [http://localhost:3000/dashboard](http://localhost:3000/dashboard) to see the dashboard.

### Run the tests

````
npm test
````
